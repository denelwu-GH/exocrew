# Lessons

Record reusable failures and their prevention here.

Each lesson should contain:

- the pitfall and its visible symptom
- the underlying cause
- why existing checks did not stop it
- the immediate response
- the durable prevention: test, gate, decision, or runbook
- evidence that the prevention works

When the same pitfall recurs, promote the prevention into a hard gate or durable decision instead of adding another narrative note.
