## One-sentence position

Models provide intelligence. Coding agents execute. Spec frameworks structure intent. CI/CD platforms run pipelines. **ExoCrew governs how work moves from a request to verified, reversible delivery.**

ExoCrew is packaged and validated for Codex today. References to Claude Code, Cursor, GitHub Copilot, OpenCode, Harness.io, Spec Kit, OpenSpec, BMAD, and Superpowers explain the surrounding ecosystem; they do not claim affiliation or unverified native integration.
